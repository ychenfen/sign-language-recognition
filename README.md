# 手语识别学习应用

基于AI的实时手语识别与学习系统，支持多种手势识别和数据收集功能。

## 🌟 功能特色

### 👥 双端分离（用户端 / 管理端）
- **用户端**：手语词典、手语学习、手语测试、实时识别、社区交流、AI 助手、个人中心
- **管理端**：用户管理、词汇管理、课程管理、社区内容管理、敏感词管理、数据概览
- **数据联动**：管理端维护的词汇/课程/社区/敏感词会同步到用户端页面

### 🎯 实时手势识别
- **10种基础手势**: 数字1-5、点赞、OK、你好、拳头等
- **MediaPipe技术**: 基于Google MediaPipe的高精度手部检测
- **实时反馈**: 即时显示识别结果和置信度
- **可视化效果**: 绘制手部关键点和连线

### 📊 数据收集中心
- **���能收集**: 自动记录手势数据用于模型训练
- **进度跟踪**: 实时显示收集进度和质量评估
- **多格式导出**: 支持JSON、CSV格式数据导出
- **统计分析**: 提供详细的数据收集统计图表

### 🎨 用户界面
- **现代化设计**: 基于Element Plus的美观界面
- **响应式布局**: 适配不同屏幕尺寸
- **镜像模式**: 支持摄像头镜像切换
- **实时控制**: 开始/停止识别等便捷操作

## 🚀 快速开始

### 环境要求
- Node.js 16+
- 现代浏览器（支持WebRTC和WebGL）

### 安装依赖
```bash
npm install
```

### 配置环境变量（启用百度AI可选）
```bash
cp .env.example .env
```

在 `.env` 中填写：
- `VITE_BAIDU_API_KEY`
- `VITE_BAIDU_SECRET_KEY`
- `DASHSCOPE_API_KEY`（如需后端AI问答）

### 启动开发服务器
```bash
npm run dev
```

### 启动后端识别服务（可选）
```bash
cd backend
python sign_recognition_api.py
```

### 默认管理员账号（本地演示）
- 用户名：`admin`
- 密码：`admin123`
- 建议首次登录后在本地存储用户列表中修改为你自己的密码

### 构建生产版本
```bash
npm run build
```

## 🎮 使用指南

### 1. 开始手势识别
1. 点击"开始识别"按钮启动摄像头
2. 将手放在摄像头前
3. 做出支持的手势动作
4. 查看识别结果和置信度

### 2. 收集训练数据
1. 选择要录制的手势类型
2. 点击"开始录制"按钮
3. 按照提示做出相应手势
4. 系统自动收集100帧数据
5. 完成后导出数据用于模型训练

### 3. 支持的手势
- **数字手势**: 1、2、3、4、5
- **常用手势**: 点赞👍、OK👌、你好👋、拳头✊
- **更多手势**: 持续扩展中...

## 📊 手势识别原理

### MediaPipe手部检测
- 检测21个手部关键点
- 实时追踪手部动作
- 支持双手同时识别

### 规则引擎识别
- 基于手指状态判断（伸直/弯曲）
- 手部朝向和位置分析
- 指尖距离和角度计算

### 识别流程
```
摄像头输入 → 手部检测 → 特征提取 → 规则匹配 → 结果输出
```
