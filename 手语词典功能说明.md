# 手语词典功能说明

## 🎯 功能概述

手语词典模块是我们应用的核心学习功能，提供了完整的手语词汇查询、学习和练习功能。该模块基于你提供的手语分解步骤图片，创建了一个丰富的手语学习资源库。

## 📚 主要功能

### 1. 词汇库管理
- **分类浏览**: 支持数字、日常用语、情感表达、称谓等分类
- **搜索功能**: 支持按词汇和拼音搜索
- **详细信息**: 包含词汇、拼音、分类、难度等信息

### 2. 详细学习页面
- **视频演示**: 支持手语视频播放（可上传）
- **分解步骤**: 基于图片的步骤化学习
- **使用说明**: 提供实际使用场景和例句
- **相关词汇**: 推荐相关手语词汇

### 3. 学习辅助功能
- **开始练习**: 直接跳转到练习模式
- **收藏功能**: 收藏重要词汇便于复习
- **学习计划**: 将词汇加入个人学习计划
- **分享功能**: 分享学习内容

## 🎨 界面设计

### 主界面布局
- **搜索栏**: 顶部搜索框，支持实时搜索
- **分类标签**: 支持按分类筛选词汇
- **词汇网格**: 卡片式展示，包含图标和基本信息

### 详情页面设计
- **基本信息区**: 词汇、拼音、分类、难度等
- **视频演示区**: 大尺寸视频播放器
- **分解步骤区**: 图片+文字的步骤说明
- **使用说明区**: 实际应用例句
- **相关词汇区**: 推荐相关学习内容
- **操作按钮区**: 练习、收藏、学习计划、分享

```javascript
  icon: '👋',              // 表情图标

  learnCount: 156,         // 学习次数统计

  steps: [                 // 分解步骤
    {
      title: '准备姿势',
      description: '手掌自然张开，五指并拢',
      imageUrl: '图片路径',
      tips: '保持手腕放松'
    }
  ],
  usage: [                 // 使用例句
    {
      sentence: '你好，很高兴认识你！',
      pinyin: 'nǐ hǎo, hěn gāoxìng rènshi nǐ!',
      meaning: 'Hello, nice to meet you!'
    }
  ],
  relatedWords: ['再见', '谢谢', '早上好']  // 相关词汇
}
```

## 🎯 交互功能

### 1. 搜索功能
- **实时搜索**: 输入即搜索，无需点击按钮
- **多字段匹配**: 支持词汇和拼音搜索
- **分类筛选**: 按类别快速筛选词汇

### 2. 详情查看
- **点击词汇卡片**: 打开详情弹窗
- **视频控制**: 支持播放、暂停、全屏
- **图片查看**: 支持步骤图片放大查看
- **例句展示**: 中文、拼音、英文对照

### 3. 学习操作
- **开始练习**: 一键跳转到练习模式
- **添加收藏**: 收藏重要词汇
- **学习计划**: 加入个性化学习计划
- **内容分享**: 分享到社交媒体
