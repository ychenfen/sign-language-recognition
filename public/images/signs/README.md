# 手语资源目录

## 资源获取方式

### 1. 官方APP录屏（推荐）
下载"国家通用手语词典"APP，录制手语视频后放入此目录

### 2. B站视频下载
使用工具下载B站手语教学视频：
- 搜索关键词：国家通用手语、手语教学
- 推荐UP主：手语课堂、无声世界

### 3. 学术数据集申请
- CSL数据集（中科大）：http://home.ustc.edu.cn/~pjh/openresources/cslr-dataset-2015/index.html
- DEVISIGN数据集：需向微软亚洲研究院申请

## 文件命名规范

视频文件：`词汇拼音.mp4`
图片文件：`词汇拼音.png`

示例：
- nihao.mp4 → 你好
- xiexie.mp4 → 谢谢
- duibuqi.mp4 → 对不起
